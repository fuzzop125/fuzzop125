How does FREE JSON host work?

What does the tool do?

It is a tool which hosts jsons. If you want a url which provides a specific JSON for testing, experimentation, hackathon or simply to test something, you can host the json here.

What is beta signup for then?

It is the next in line tool called Instamock which we will release soon. It takes out hassle of different domains and let you mock APIs and more directly from your localhost/staging with no rate limits. We will notify your for beta in March 2023.

How does it work?

It is simple. You enter a json in the box and submit it. A popup will come telling the path of the hosted json. In the list above, you can view your past jsons you created along with the path. Edit is not allowed.

What is the path of the hosted json?

For example, your json is {"key" : "value", "key2" : ["x", "y"]} and you receive path as https://app.fuzzop.com/json/1q23e4r. Here are the possibilities.

PathResponse
https://app.fuzzop.com/json/1q23e4r{"key" : "value", "key2" : ["x", "y"]}
https://app.fuzzop.com/json/1q23e4r/key"value"
https://app.fuzzop.com/json/1q23e4r/key2"["x", "y"]"
https://app.fuzzop.com/json/1q23e4r/key2/1"x"

You should be able to fetch jsons without trouble from host of your local/staging.

What are the limits?

There are certain limits of course being a free tool. First of all, the max json size is 10kb and the max it will be hosted for is 12 hours. Lastly, the rate limit is 100rpm beyond which you will be stopped for a while. Otherwise, create as many jsons as you like.

What are the liabilities we take as json host?

Zero. While we maintain our best to keep serving awesome jsons, we don't intend to take any responsiblity of what you are hosting and how it is being used. If you face any financial loss because of this, we are not responsible. In short, we won't take hits for troubles you face.
